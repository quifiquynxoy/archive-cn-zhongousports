# archive-cn-zhongousports

## 项目简介

本项目是一个面向中文网络空间中体育类独立 HTML 页面的归档仓库。主要目的是对特定时间段内可获取的、以独立 HTML 文件形式呈现的体育相关内容进行收集、整理与存档，不针对任何单个域名或具体网站进行定向镜像或抓取。

本仓库仅作为静态文件存档使用，不提供任何形式的实时更新服务，也不对归档内容进行编辑、修改或二次加工。

## 目录说明

```
.
├── README.md          # 本文件
├── pages/             # 归档的 HTML 页面存放目录
│   ├── 2024/          # 按年份划分的子目录
│   └── ...
└── meta/              # 元数据或辅助说明文件（可选）
```

- `pages/`：按照年份组织，存放归档的独立 HTML 文件。文件名通常保留原始页面标题或时间戳，以便检索。
- `meta/`：可选的元数据目录，用于存放归档清单、校验信息或其他辅助文档。

## 页面归档说明

- 归档内容均为原始获取的 HTML 文件，未经过内容修改或格式转换。
- 归档文件可能包含外部资源引用（如图片、样式表、脚本），这些资源可能已不可用或仅在原始环境中有效。
- 归档时间以文件提交到仓库的时间为准，不保证与页面实际发布时间一致。
- 归档范围限于作者在合法范围内能够获取的公开内容，不包含需要登录、付费或特殊权限访问的页面。
